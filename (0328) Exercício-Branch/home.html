Branch
Início Definição Como Fazer

Introdução: Quando criamos um código curto, encontrar e corrigir erros é uma tarefa simples e dinâmica. Porém, conforme o código torna-se mais robusto, mais complicações ao alterá-lo e comentá-lo surgem.
Para facilitar esse processo, podemos utilizar o Git.

Git: Criado em 2005 por Linus Torvald, o Git é um sistema de controle de versão projetado para lidar com pequenos ou grandes projetos, agindo de maneira veloz e eficiente.
Através dele, é possível alterar e armazenar versões do código, rastrear quem fez alterações e codar de forma colaborativa, além de outras funções (que podem ser visualizadas aqui).

O sistema Git possui diversos comandos. Entre eles, a branch.
